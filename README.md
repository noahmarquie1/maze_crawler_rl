## Maze Crawler for Kaggle

Competition entry by Noah Marquié and Michael Dickinson

### Usage

Configure Environment - Recommended Python 3.12.0
```
pip install -e .
```

Train Agent
```
python agent.py
```

Run model

```
python main.py
```
